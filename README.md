# Laptop Price Prediction

## Overview
Machine Learning project to predict laptop prices using different laptop specifications like brand, RAM, processor, storage, and screen size.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Matplotlib
- Seaborn
- Joblib

## Features
- Predict laptop prices
- User-friendly interface
- Data preprocessing
- ML model training
- Real-time prediction
- Interactive web application
- Brand-wise price prediction
- Fast and accurate results
- Responsive UI design
- Model saving and loading
- Data visualization
- Easy deployment

## Dataset Information
- Laptop brand data
- RAM and storage details
- Processor information
- Operating system data
- Price dataset for training

## Machine Learning Algorithms
- Linear Regression
- Random Forest Regressor
- Decision Tree Regressor

## Project Structure
- app.py
- model.pkl
- dataset.csv
- requirements.txt
- README.md

## Advantages
- Saves time for users
- Helps compare laptop prices
- Simple and easy to use
- Useful for students and beginners
